# sum11crawler

## Grab SUM-11 index

```bash
go run main.go grab-index
```

## Download SUM-11 articles

```bash
go run main.go grab-articles
```

## Grab SUM-11 user`s index

```bash
go run main.go grab-index -v sum.in.ua_user
```

## Download SUM-11 user`s articles

```bash
go run main.go grab-articles
```
